How do the rankings work?

I created the rankings using the following variables: How hard is the bug to treat, how likely will the bug develop resistance, how common is the bug, and how fatal is the bug (assuming appropriate treatment). Rank 1 is the “hardest” to beat, rank 16 the “easiest” to beat.

Each person shooting for each bug will shoot using different distances from the hoop and different “ball” types to vary how challenging it is to score. I’m going to leave it up to the individuals how far away from the basket they want to shoot and which bug/ball they want to shoot with, so there can absolutely be upsets!
